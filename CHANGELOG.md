## Release Notes

### v1.0.0 (2026-05-14)

Initial release with core functionality:

**Features:**
- Search notes in IMA knowledge base
- Import individual notes by ID
- Browse all knowledge bases
- Configure IMA credentials in settings
- Auto-sync option (coming soon)

**Known Issues:**
- Sidebar panel not yet implemented (currently shows "Coming Soon")
- No bidirectional sync (Obsidian → IMA)
- Credentials must be manually copied from QClaw (no automatic token refresh)

**Next Steps:**
- Implement sidebar panel for browsing IMA notes
- Add note preview functionality
- Support bidirectional sync
- Add incremental sync (only sync updated notes)
- Improve UI with better styling
